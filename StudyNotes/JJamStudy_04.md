# Operator
- 기본 연산
``` python
print(a + b)    # 덧셈
print(a - b)    # 뺄셈
print(a * b)    # 곱셈
print(a / b)    # 나눗셈
e = 3
print(a // e)   # 몫
print(a % e)    # 나머지
print(a ** 2)   # 거듭제곱
```

# Operator
- 논리 연산
```python
a = True
b = False

print(a and b)

```

# Operator
- 비교 연산
```python
a = 100
```
